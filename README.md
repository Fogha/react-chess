# React Chess Web App

A 2 player chess game built in react.

## Tech Stack

**Client:** ReactJs, RxJs, Bulma, React-router-dom, Chess.Js.

**Server:** Firebase.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Fogha/react-chess.git
```

Go to the project directory

```bash
  cd react-chess
```

Install dependencies

```bash
  npm install
```

Start react app

```bash
  npm start
```
